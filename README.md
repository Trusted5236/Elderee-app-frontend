# Elderee App - Frontend

This is the frontend of the **Elderee App**, built using **React Native + Expo Router**. It handles the mobile user interface for onboarding, authentication, and in-app features.

## 🚀 Tech Stack

- **React Native** (with Expo)
- **Expo Router** (file-based navigation)
- **NativeWind** (Tailwind CSS for React Native)
- **TypeScript**
- **Git + GitHub for version control**
